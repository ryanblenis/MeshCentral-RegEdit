## [0.0.2] - 2025-03-06

### Added

- Default key visibility (mirroring RegEdit view) - @stephannn
- Alphabetical ordering of keys/values - @stephannn
- Remove "-" symbol if no child items

## [0.0.1] - 2025-02-27

### Added

- Initial release of MeshCentral-RegEdit plugin
- Browse registry hives (HKLM, HKCU, HKCR, HKU, HKCC)
- View registry values with type and data display
- Edit existing registry values (name, type, data)
- Create new registry keys and values
- Rename registry keys and values
- Delete registry keys (with recursion) and values
- Export registry branches to .reg format
- Right-click context menu for Edit, Rename, Delete, Refresh actions
- Toolbar buttons for quick actions
- Import and Export .reg files
